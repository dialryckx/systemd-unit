This is a systemd-unit for http server on Python.
1. Automatically launches when the PC turns on
2. Restarts when crashed
3. Memory limits
4. After kill -9 <PID> the server starts automatically

Installation Guide for Linux/MacOS:
1. `git clone https://github.com/dialryckx/systemd-unit`
2. `cd systemd-unit/`
3. `chmod +x install-service.sh`
4. `./install-service.sh`
